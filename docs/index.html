c-natural

Made by Ahmer Raza, Madeline Woehrman, Elliot Phillips, and Stephen Becker

A programming language for learners focused on providing easy-to-read, obvious syntax, while still being able to convey intermediate- and advanced-level programming topics.

Covered concepts include:

  • Primitive data types
  • Logic and conditionals
  • Arithmetic
  • Arrays
  • Loops
  • Functions
  • structs

Basic data types

Basic data types include

  • integer (int)
  • float (float)
  • character (char)
  • double (double)

Typecasting is also available.

Example code:

# implicit declaration + assignment
declare whole number x
set x to 11

# explicit declaration + typecasting
set x to 11 as a character
set y to x as an integer

Logic and Conditionals

Logical and relational operators; the phrasing of a logical operator changes when it is in a if statement vs when on it's own. There are six logical operators:

  • is equal to
  • is not equal to
  • is greater than
  • is less than
  • is less than or equal to
  • is greater than or equal to
is x equal to y
if x is equal to y,
    do this...
otherwise, if x is less than y,
    do that...
otherwise,
    do this other thing...

Arithmetic

Arithmetic operators include

  • plus
  • minus
  • times
  • divided by
  • modulo
  • and (&&)
  • or (||)
  • not (!)
  • AND (&)
  • OR (|)
  • XOR (^)
  • NOT (~)
  • shifted left by (<<)
  • shifted right by (>>)

Example:

set x to 9 shifted left by 2
set y to x XOR 3 minus 4 divided by 7

Arrays

Arrays can be one-dimentional or two-dimentional. The sort keyword sorts the given array through bubble sort.

define Array nums of whole numbers of size 10

get element at 5 of nums
set element at 7 of nums to 15

define 2DArray grid of characters of size 5 by 5

get element at 2, 4 of grid
set element at 1, 3 of grid to '*'

Functions

Functions are declared using the declare keyword:

declare void function foo
    print 6
end function declaration

add parameters to a function by adding them in the declaration:

declare integer function foo with integer x, integer y
    print x
    print y
end function declaration

To call a function, use the call keyword:

call foo with 5, 6
